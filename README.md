# fuels-ts

Fuel v2 TypeScript SDK

## Setup

```sh
npm install
npm run build
```

## Test

```sh
# run all tests
npm run test
# run tests and get coverage
npm run test:coverage
```

## Contributing

Code must be formatted and linted.

```sh
npm run prettier-format
```
