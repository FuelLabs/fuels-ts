# fuels-ts
Fuel v2 TypeScript SDK

## Building

Install dependencies:

```sh
npm install
```

Build and run tests:

## Test

```sh
 npm run test // run all tests
 npm run test:coverage // run tests and get coverage
```

## Contributing

Code must be formatted and linted.

```sh
npm run prettier-format
```
