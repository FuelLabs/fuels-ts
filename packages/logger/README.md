# `@fuel-ts/logger`

**@fuel-ts/logger** is a sub-module for logging within the **Fuel** TS ecosystem.

A logging component that can be used to log messages to the console or to a file.

```sh
pnpm add @fuel-ts/logger
# or
npm add @fuel-ts/logger
```
